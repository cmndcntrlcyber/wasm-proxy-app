WASM Proxy Application

This application executes and proxies the rust-run binary output.

Execute rust-run Application

When executed, this application will:

  1. Execute the rust-run binary located at ../rust-run/target/release/rust-run.exe
  2. Proxy the execution through a WebSocket bridge
  3. Display real-time output from the rust-run application
  4. Provide execution status and results
